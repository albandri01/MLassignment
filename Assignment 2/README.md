Dataset Name: Laptop Price Dataset
Dataset Source: This dataset was obtained from Kaggle
The dataset contains approximately: 1303 rows and 13 columns
Columns Description:
• Company: Brand of the laptop (e.g., Dell, HP, Apple)
• Product: Model name of the laptop
• TypeName: Type/category of the laptop (Notebook, Ultrabook, Gaming,
etc.)
• Inches: Screen size of the laptop in inches
• ScreenResolution: Display resolution of the laptop
• Cpu: Processor information
• Ram: Amount of RAM in GB
• Memory: Storage type and size
• Gpu: Graphics card information
• OpSys: Operating system
• Weight: Weight of the laptop in kilograms
• Price: Price of the laptop (target variable)
The purpose of this dataset is to perform exploratory data analysis (EDA) to
understand the distribution of laptop prices and compare prices across different
companies, Analyze the relationship between price and features such as RAM,
weight, and screen size and lastly to Identify patterns and trends in the dataset
